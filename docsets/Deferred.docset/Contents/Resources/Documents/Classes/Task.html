Task

public final class Task<SuccessValue>: NSObject

A wrapper over any task.

Forwards operations to an arbitrary underlying future having the same result type, optionally combined with some cancellation.

  • ignored()

    Returns a task that ignores the successful completion of this task.

    This is semantically identical to the following:

    myTask.map { _ in }

    But behaves more efficiently.

    The resulting task is cancellable in the same way the receiving task is.
